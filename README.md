# ExpenseWise - Smart Expense Tracker

ExpenseWise is a modern, intuitive expense tracking application built with Next.js and ShadCN UI. It provides a clean dashboard to visualize your income, expenses, and budget, along with AI-powered insights to help you manage your finances better.

## üöÄ Getting Started

To get the application up and running on your local machine, follow these steps.

### Prerequisites

-   Node.js (v18 or later)
-   npm or yarn

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/expensewise.git
    cd expensewise
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```
    or
    ```bash
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:9002`.

## üìÇ Project Structure

-   `src/app/`: Contains the main pages and layouts of the application, following the Next.js App Router structure.
-   `src/components/`: Includes all the reusable React components, organized by feature (e.g., `dashboard`, `ui`).
-   `src/lib/`: Holds utility functions, mock data, and type definitions.
-   `src/context/`: Contains React context providers for global state management (e.g., user data).
-   `src/ai/`: Contains Genkit flows for AI-powered features.
-   `public/`: Stores static assets like images and fonts.
-   `styles/`: Global styles and Tailwind CSS configuration.
