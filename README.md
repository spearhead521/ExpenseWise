# ExpenseWise - Smart Expense Tracker

ExpenseWise is a modern, intuitive expense tracking application built with Next.js and ShadCN UI. It provides a clean dashboard to visualize your income, expenses, and budget, along with AI-powered insights to help you manage your finances better.

## ‚ú® Features

-   **Dashboard Overview**: A comprehensive summary of your financial status, including total income, expenses, and current balance.
-   **Transaction Management**: Easily log and view all your income and expense transactions. Filter them by type (all, income, expense).
-   **Budget Tracking**: Set monthly and category-specific budgets to keep your spending in check. Visualize your progress with clear progress bars.
-   **Financial Reports**: In-depth reports with charts to analyze your income vs. expense trends and spending by category.
-   **AI-Powered Insights**: Get smart, actionable insights into your spending habits, generated by AI.
-   **User Profile Settings**: Customize your profile, including your name, email, and profile picture.
-   **Light & Dark Mode**: Switch between light and dark themes for your preferred viewing experience.
-   **Responsive Design**: A fully responsive interface that works seamlessly on desktop and mobile devices.

## üöÄ Getting Started

To get the application up and running on your local machine, follow these steps.

### Prerequisites

-   Node.js (v18 or later)
-   npm or yarn

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/expensewise.git
    cd expensewise
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```
    or
    ```bash
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:9002`.

-   `src/app/`: Contains the main pages and layouts of the application, following the Next.js App Router structure.
