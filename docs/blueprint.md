# **App Name**: ExpenseWise

## Core Features:

- User Authentication: Secure sign-up and login functionality with password encryption and 'forgot password' option.
- Dashboard: Display a summary of total expenses and income, recent transactions, and a pie chart of spending categories.
- Add Expense/Income: Enable users to add new income and expense records.
- Edit & Delete Transactions: Allow users to update or remove existing transaction records.
- Categories Management: Enable users to add and manage expense categories with customizable icons and colors.
- Monthly Budget: Let the user set a monthly budget with a progress bar, and send alerts when the user reaches 50%, 75%, 100% of it.
- Smart Expense Insights: Leverage AI to provide users with insights into their spending habits, such as overspending warnings or increased expense notifications. Use it as a tool for when to alert users based on their transaction history

## Style Guidelines:

- Primary color: A calming teal (#4DB6AC) to inspire financial zen.
- Background color: A soft, desaturated teal (#E0F2F1) to soothe the eyes.
- Accent color: A vibrant, contrasting orange (#FF7043) to draw attention to key actions and alerts.
- Body and headline font: 'Inter', a grotesque-style sans-serif, will be used for a modern, objective feel.
- Use clear, minimalist icons for categories and actions.
- Design a responsive layout that adapts to mobile, tablet, and desktop screens.
- Implement subtle transitions and animations to enhance user experience.
